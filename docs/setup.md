# セットアップガイド

## 必要な環境

- Node.js 18.x 以上
- Go 1.20 以上

## インストール

### フロントエンド

```bash
cd frontend
npm install
npm start
```

### バックエンド

#### Node.js版

```bash
cd backend
npm install
node server.js
```

#### Go版

```bash
cd backend
go run main.go
```

## 開発環境

1. リポジトリをクローン
2. 依存関係をインストール
3. 開発サーバーを起動

## テスト

```bash
npm test
```
